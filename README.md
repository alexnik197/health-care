# 🧘‍♂️ Health Care App

**Health Care** — простое элегантное приложение для заботы о здоровье во время работы за компьютером. Сидит в фоновом режиме (в трее) и периодически напоминает сделать полезную разминку для глаз и шеи.

Создано с любовью для тех, кто много времени проводит за монитором! 💙

---

## ✨ Что внутри?

- **Умные напоминания:** Приложение само напомнит, когда пора отвлечься от экрана (по умолчанию — раз в 60 минут).
- **Комплекс упражнений:** Встроенные проверенные упражнения:
  - 👁 *Для глаз* — фокусировка, круги, моргание, зажмуривание.
  - 🦒 *Для шеи* — ретракция, наклоны с дотяжкой, взгляд в карман.
  - 🏋️‍♂️ *Дополнительные* — вращение плечами для раскрытия грудной клетки.
- **Удобный таймер:** Каждое упражнение имеет визуальный таймер, который автоматически переключает на следующий этап.
- **Гибкие настройки:** Настраивайте время каждого упражнения, частоту уведомлений и автозапуск с системой.
- **Сворачивание в трей:** Приложение тихо работает в системном трее, не мешая в панели задач.

## 🚀 Как запустить?

Работает на Windows, Linux и macOS. Стек: **Electron + Vue 3 + TypeScript**.

### Требования
[Node.js](https://nodejs.org/) 18+.

### Установка и запуск (разработка)

```bash
git clone https://github.com/alexnik197/health--care.git
cd health--care
npm install
npm run dev
```

> **Отладка:** В dev-режиме нажмите `F12` для открытия DevTools.

### 📦 Сборка

```bash
npm run build
```

Готовые файлы (`.AppImage`, `.snap` для Linux) появятся в папке `release/`.

**На Linux запускать с флагом:**
```bash
./release/health-care-1.0.0.AppImage --no-sandbox
```

│   ├── exercise.ts     # Exercise, ExerciseCategoryMap, ExerciseCategoryKey
│   ├── settings.ts     # PersistedSettings

## 🛠 Технологии

| Технология | Назначение |
|---|---|
| **Vue 3** (Composition API + `<script setup lang="ts">`) | UI |
| **TypeScript** | Строгая типизация всего кода |
| **Pinia** | Реактивное управление состоянием |
| **Electron** | Нативные уведомления, трей, автозапуск |
| **Vite** | Сборка и dev-сервер |
| **Vanilla CSS** | Glassmorphism, плавные анимации |
