# 🧘‍♂️ Health Care App

**Health Care** — это простое и элегантное приложение для заботы о вашем здоровье во время работы за компьютером. Оно сидит в фоновом режиме (в трее) и периодически напоминает вам сделать полезную разминку для глаз и шеи. 

Создано с любовью для тех, кто много времени проводит за монитором! 💙

---

## ✨ Что внутри?

- **Умные напоминания:** Приложение само напомнит, когда пора отвлечься от экрана (по умолчанию — раз в 60 минут).
- **Комплекс упражнений:** Встроенные и проверенные сеты:
  - 👁 *Для глаз* (фокусировка, круги, моргание, зажмуривание).
  - 🦒 *Для шеи* (ретракция, наклоны, повороты).
  - 🏋️‍♂️ *Бонус* (вращение плечами для раскрытия грудной клетки).
- **Удобный таймер:** Каждое упражнение имеет свой визуальный таймер, который заботливо переключит вас на следующий этап.
- **Гибкие настройки:** Настраивайте время каждого упражнения под себя, меняйте частоту уведомлений и включайте автозапуск вместе с системой.
- **Сворачивание в трей:** Приложение не мешает в панели задач, тихо работая в системном трее вашей операционной системы.

## 🚀 Как запустить у себя?

Приложение работает на Windows, Linux и macOS. Для разработки используется стек **Electron + Vue 3**.

### Требования
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18+).

### Установка и запуск (для разработки)
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/alexnik197/health--care.git
   cd health--care
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```

### 📦 Сборка готового приложения
Чтобы собрать готовый установочный файл (билд) для вашей операционной системы, просто выполните:
```bash
npm run build
```
Готовые исполняемые файлы (например, `.exe` для Windows, `.AppImage` или `.snap` для Linux) появятся в папке `dist/`.

## 🛠 Технологии

Мы ценим легкость и чистоту кода, поэтому не используем тяжелые UI-библиотеки:
- **[Vue 3](https://vuejs.org/) (Composition API)** — для создания динамичных и быстрых интерфейсов.
- **[Pinia](https://pinia.vuejs.org/)** — для реактивного управления состоянием таймеров и настроек.
- **[Electron](https://www.electronjs.org/) + Vite** — для связи интерфейса с операционной системой (уведомления, трей, автозапуск).
- **Vanilla CSS** — премиальный дизайн с элементами glassmorphism (размытие фона) и плавными современными анимациями.
